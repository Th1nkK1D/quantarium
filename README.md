# Quantarium

Meta-repo for Quantarium project

## Instructions

1. Clone or download this repo
2. Install required packages
   ```bash
   npm i
   ```
3. **Update** child repositories
   ```bash
   npm run update
   ```
4. **Build** child repositories that required
   ```bash
   npm run build
   ```
5. **Start** all services
   ```bash
   npm run start
   ```
